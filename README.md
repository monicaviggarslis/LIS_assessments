# Right Word – Assessment 2

This repository contains the notebook and supporting files for **Right Word Assessment 2**.

## Project Environment

This project was developed and tested using **Python 3.12.12**.

To install the required Python libraries, run:

pip install -r requirements.txt

## API Configuration

The notebook uses the **OpenAI API**.  
For security reasons, the API key is **not** included in this repository.

To enable API‑related functionality:

1. Create a file named `.env` in the **root directory** of the project.
2. Add the following line to the file:

OPENAI_API_KEY=your_api_key_here

A sample file named `.env.example` is included to show the correct format.  
The `.env` file itself should **not** be committed to GitHub.

## Repository Contents

- The main assessment notebook  
- Supporting files required for running the notebook  
- A `requirements.txt` file listing the necessary Python dependencies
